# 2026 03 28 REV1.3 Install and Dependency Notes

## Current baseline
- One local `.venv`
- Python 3.12.x
- `install_project.ps1` is the canonical baseline installer for the current local workflow

## Important notes
- `install_project.ps1` should create `.venv` if it does not exist.
- `rebuild_venv_py312.ps1` is the destructive reset path for a fresh `.venv` rebuild.
- PyBullet remains optional / deferred. If MSVC is not ready, the install should continue with the fallback-capable baseline.
- Torch remains in the install script for now. Whether it should remain part of the default baseline is still open.

## Preferred local commands
```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\windows\install_project.ps1
powershell -ExecutionPolicy Bypass -File .\scripts\windows
un_poc1_validation.ps1 -BackendMode numpy
```

## Destructive rebuild
```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\windows
ebuild_venv_py312.ps1
```

## Clean full snapshot
Use the 7-Zip-based snapshot script:
```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\windows\export_snapshot.ps1 -Revision 0.0.8
```
