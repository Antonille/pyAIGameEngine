2. search for similar architectures, batching strategies, runtime layouts, and tooling paths
3. separate findings into recommended-now / plausible-later / deferred
4. update lessons, workflow, prompts, and resource docs
5. narrow the next step into either an architecture/spec pass or an implementation pass
