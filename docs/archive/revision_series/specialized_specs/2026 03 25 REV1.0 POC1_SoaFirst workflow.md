# POC1_SoaFirst Workflow — REV1.0

## Purpose
POC1 validates the first-choice architecture stack on the current single-venv Python 3.12 baseline:
- NumPy SoA engine state
- PyBullet backend boundary
- pyglet + ModernGL viewer
- minimal Gym-compatible wrapper
- timing/profiling harness

## Isolation rules
- Keep POC1 self-contained.
- Do not import future shared modules from root until they are explicitly promoted.
- If a utility looks universally useful, document it first, then promote in a later revision.

## Current targets
1. Confirm local Python 3.12 install works.
2. Confirm viewer starts.
3. Confirm benchmark runs headless.
4. Confirm Gym wrapper returns stable observations and rewards.
5. Add engine-loop stage naming and timing breakdown.

## Local validation order
1. `scripts/windows/install_project.ps1`
2. `scripts/windows/run_poc1_validation.ps1`
3. `POC1_SoaFirst/scripts/run_viewer.py`
