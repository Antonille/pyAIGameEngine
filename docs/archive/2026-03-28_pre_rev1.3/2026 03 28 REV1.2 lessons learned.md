# 2026 03 28 REV1.2 lessons learned

1. Native-physics dependency issues should not block architectural progress.
2. Validation scripts must fail hard when benchmark or rollout substeps fail.
3. Small isolated POC deltas are useful, but periodic full-snapshot consolidation is necessary.
4. Early stage profiling is worth it; it surfaced event-processing as the dominant bottleneck.
5. AI inference architecture should be framed around model-family scheduling, stale-tolerance, and batching, not around per-agent every-cycle execution.
6. Repeated object families benefit from fixed runtime schemas and precomputed feature-pack plans.
