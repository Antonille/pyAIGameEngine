# Install and Dependency Notes — REV1.0

## Local assumptions
- Path: `C:\PythonDev\Dev1\pyGames\pyAIGameEngine`
- Python baseline: **3.12.x**
- Use a **single** local `.venv`
- `git` and `gh` installed
- `uv` optional, not required by the Windows baseline script

## Baseline package set for current work
- core numeric: `numpy`
- physics: `pybullet`
- rendering/windowing: `pyglet`, `moderngl`
- RL wrapper compatibility: `gymnasium`
- profiling/benchmark support: standard library now; optional tools later
- kernel experiments (next step): `numba`
- AI baseline (same venv for now): `torch`, `torchvision`, `torchaudio`

## Installation approach
Use the root install script first. It rebuilds or creates `.venv` on **Python 3.12**, bootstraps packaging tools, installs POC1 in editable mode, and installs the baseline package set.

## Important rule
If the existing `.venv` is not on Python 3.12, remove it and rerun the install script. Do not keep multiple project venvs for now.
