## Current architecture truth
1. Runtime hot-path data should favor **SoA** layouts.
2. POC1 remains **fallback-first** and should not be blocked by PyBullet.
3. AI inference should not be tied to every agent every plank-tic.
4. Repeated object families should use **fixed runtime schemas** and **feature-pack plans** rather than dynamic field fetching.
5. AI scheduling should operate on **model families**, cadence, freshness, and batchability.
6. Rendering remains 2D for now, but simulation state remains fully 3D.
7. Simulation-bearing scene frames are rigid and inertial in current POC assumptions.
8. Body-local / shape-local frames are rigidly attached to bodies, are generally non-inertial, and are used for geometry/material description rather than primary force-balance solving.
9. Master gravity is a time-varying universal vector owned by the master stationary reference frame and propagated to child simulation frames.
10. First-pass field coupling is center-of-mass based, with a staged upgrade path to distributed interaction samples.
