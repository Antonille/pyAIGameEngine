## Architecture decisions active in this pass
1. Simulation remains fully 3D while rendering stays 2D for now.
2. Shape-local/body-local frames may be non-inertial and are used for geometry/material description; active simulation frames remain the primary dynamics frames.
3. Master gravity is a universal time-varying vector owned by the master frame.
4. Child simulation frames use rigid-body transform propagation and inherited gravity revision tracking.
5. First-pass field coupling treats bodies as center-of-mass point interactions.
6. Primitive collision/response architecture is beginning with explicit contact/manifold data structures rather than dense voxel collision.
