# REV4.6 docs update notes

This sparse refresh updates documentation only.

Main themes:
- GitHub main has already absorbed the REV4.5 cleanup promotion.
- Canonical environment-variable names are now explicit.
- Historical revisions-path aliases are retired.
- Durable repo files should be transferred by artifacts or ZIPs, not raw chat copy/paste.
- Repo-owned line-ending policy should be implemented through `.gitattributes`.
