The most effective prompts in this project usually separate:
1. current-truth anchor
2. pass type
3. constraints / deferrals
4. required outputs
5. exact validation expectations
6. next-step handoff

## Research sweep template
Use the current pyAIGameEngine project state as truth.

Pass type:
Research sweep.

Check whether the current architecture path is still well-supported by outside evidence before widening the implementation branch.

Requirements:
1. Review the current-pointer docs first.

Pass type:
Implementation pass.

Task:
Implement the first working slice of the long-lifecycle integrated testing and reporting system for pyAIGameEngine.

Execution rule:
While executing, evaluate whether the currently designed plan remains optimal. If a better design is discovered during implementation, update the plan, document the change, and implement the improved design as documented.

Core implementation requirements:
1. Implement a modular but integrated test harness.
2. Implement an append-only archival test data schema and writing path.
3. Implement apples-to-apples comparison rules and an early revision of a complexity-normalized projection method.
4. Ensure the design supports future:
   - smoke tests
   - acceptance tests
   - regression tests
   - benchmark/performance tests
   - simulation-credibility tests
   - AI/RL-facing tests
5. Create the required Python classes, methods, and source code.
6. Minimize terminal prompting required to launch the test system with desired parameters.
7. Keep archival test data as the durable source of truth.
8. Keep the current human-readable report as a regenerable artifact derived from archival data.
9. Record fixed seeds, compatibility keys, and machine/runtime metadata where practical.
10. Keep the archival/reporting design lightweight now but compatible with richer benchmark-history and experiment-tracking tools later.

Deliver:
- sparse snapshot ZIP
- updated code
- updated docs
- durable calling syntax / bash command to run the test
- exact local validation commands
- concise summary of what was implemented
