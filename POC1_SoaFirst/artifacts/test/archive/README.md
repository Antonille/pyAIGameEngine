# Append-only archival test data

This directory stores durable append-only archival test history generated by the integrated test harness.

Primary archive file:
- `test_run_archive.jsonl`

Rules:
- do not hand-edit prior run records
- new local runs append new JSON records
- current human-readable reports are regenerated from this archival data
- container-generated sample runs were intentionally excluded from this snapshot so local validated runs remain the source of truth
