# Current regenerated reports

This directory contains the latest human-readable report outputs derived from archival test data.

Expected files after a local regeneration pass include:
- `current_test_report.md`
- `current_test_report.html`
- `generated/*.svg`

These files are replaceable convenience outputs.
The durable historical truth remains the archival JSONL records and run-linked artifacts under `../../artifacts/test/`.
