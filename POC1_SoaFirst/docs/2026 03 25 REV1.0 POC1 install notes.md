# POC1 Install Notes — REV1.0

## Intended install
From repo root:
```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\windows\install_project.ps1
```

## Python baseline
- Use **Python 3.12.x**
- Use **one project `.venv` only**

## Validation order
```powershell
.\.venv\Scripts\python.exe .\POC1_SoaFirst\scripts\run_benchmark.py --steps 500 --n 128
.\.venv\Scripts\python.exe .\POC1_SoaFirst\scripts\run_gym_rollout.py --steps 64
.\.venv\Scripts\python.exe .\POC1_SoaFirst\scripts\run_viewer.py
```

## Known risk
Even on Python 3.12, local graphics/physics setup may still require machine-specific fixes. If one package fails, document the exact error in the next snapshot and keep the rest of the POC structure intact.
